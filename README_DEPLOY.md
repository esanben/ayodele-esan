## Publish on GitHub Pages

1. Create a new GitHub repository, or use `esanben.github.io` for a user site.
2. Upload every file and folder in this zip to the repository root.
3. Commit and push the files.
4. In GitHub, open Settings > Pages.
5. Set the source to the main branch and root folder.
6. Wait for GitHub Pages to build, then open the published URL.

## Optional final edits

- Review the text in `index.html` and adjust the bio tone if you want it to sound more formal, more executive, or more academic.
- Replace any generated SVG visuals in `images/visuals/` if you later prefer journal front-page screenshots.
- The website is static and does not require any build step.
